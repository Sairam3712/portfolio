# Portfolio

This is my personal portfolio website built with HTML and CSS. It showcases my background, technical skills, projects, experience, and contact information.

## Features
- Responsive layout
- Animated background
- Project links to live demos
- Resume download button

## Files
- portfolio.html — main portfolio page
- index.html — GitHub Pages redirect page
- resume.pdf — downloadable resume

## Preview
Open portfolio.html in a browser, or visit the live site on GitHub Pages.
